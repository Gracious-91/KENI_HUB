# KENI_HUB
A Digital Forensics Project Repository
Keni Hub is a personal digital forensics lab where I document my learning journey through hands-on projects in computer forensics, including data acquisition, analysis, and cybersecurity investigations
🔍 Overview

Keni Hub is a personal digital forensics lab where I document my journey in learning and applying forensic techniques through practical projects. This repository contains hands-on work focused on evidence acquisition, analysis, and cybersecurity investigations.

🎯 Objectives
To build practical skills in digital forensics
To document forensic processes and methodologies
To explore tools used in real-world investigations
To develop a strong foundation for a career in cybersecurity and digital forensics
🧰 Areas Covered

This repository includes projects in:

Computer Forensics
Disk Imaging & Analysis
File Recovery Techniques
Basic Network Analysis
Forensic Documentation
🛠️ Tools & Technologies

Some of the tools used in this repository include:

FTK Imager
Autopsy
Wireshark
Other open-source forensic tools
📁 Project Structure

Each project will typically include:

A clear objective
Tools used
Step-by-step process
Findings and results
Screenshots (where applicable)
📈 Learning Approach

This repository reflects a hands-on learning approach, where each project builds on previous knowledge. The focus is on understanding both the tools and the reasoning behind forensic procedures.
